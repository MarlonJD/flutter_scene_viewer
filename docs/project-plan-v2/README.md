# Project Plan v2

This directory is a preserved v2 planning source pack. It is not the active
execution plan set for the repository.

Use it as source material when planning later milestones. Before implementing
anything from this pack, promote the relevant slice into a small, verifiable
plan under `docs/exec-plans/active/` and make that active plan the source of
truth for the work.

The canonical public package documentation lives in the top-level `README.md`
and focused files under `docs/`. The canonical agent entry point is the root
`AGENTS.md`.
